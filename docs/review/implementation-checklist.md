   ```bash
   # プロジェクト初期化
   npm init -y
   
   # 依存パッケージインストール
   npm install -D typescript @types/vscode
   
   # TypeScript設定
   tsc --init
   ```

2. 基本実装開始
   ```typescript

       // 初期化処理

   }
   ```

   ```bash
